# How assessments are submitted (all inside the portal)

Everything is done **inside the PBK LMS portal** — students do not leave the site to submit.

---

## 1. Quizzes (Formative quiz, Module test, Final exam)

**Where:** My courses → [Course] → [Module] → **Quizzes & assessments** → click the quiz.

**Steps:**
1. Complete all units in the module (click each unit so it’s marked done).
2. The **Quizzes & assessments** section unlocks.
3. Click the quiz (e.g. “Formative Quiz Human Resource Management”).
4. On the quiz page you see each question with options **A, B, C, D**. Select one answer per question.
5. Click **Submit**. The quiz is auto-graded and your score is shown on the same page.

**Done entirely in the portal:** answer and submit on the same page; no external site, no file upload.

---

## 2. Assignments

**Where:** My courses → [Course] → [Module] → **Quizzes & assessments** → click the assignment.

**Steps:**
1. Complete all units in the module so assessments unlock.
2. Click the assignment (e.g. “Human Resource Management Assignment”).
3. Read the **Brief** and **Rubric** on the page.
4. Prepare your report/document in Word (or similar) on your computer, as per the brief.
5. On the **same** assignment page in the portal:
   - Use **“Upload your work (PDF, Word, image)”** and choose your file.
   - Click **Submit assignment**.
6. The file is uploaded and your submission is recorded **inside the portal**. No need to email or use another site.

**Submission is 100% in the portal:** only the writing is done elsewhere; upload and submit happen on this page. You do not leave the project/portal to submit.

---

## Summary

| Type        | Where you do it                    | Submit in portal? |
|------------|-------------------------------------|-------------------|
| Quiz       | Answer on the quiz page            | Yes — click Submit |
| Assignment | Write in Word, then upload on the assignment page | Yes — choose file + Submit assignment |

Both are submitted **directly in the portal**; there is no separate “submit elsewhere” step.
